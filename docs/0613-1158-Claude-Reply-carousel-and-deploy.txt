4) Core Root Cause(s)
A) Project Pages subpath issue
- Hosted under: https://kevin-413.github.io/community_band_site/
- HTML was using root-relative URLs such as /img/... and /join/... which resolve to the domain root, not the project subpath.
- Result: broken images/links and confusing visual behavior.

B) Carousel UX regressions across iterative fixes
- Captions moved out of overlay context temporarily, appearing detached.
- Autoplay behavior depended only on data-flickity attributes; some environments showed non-starting autoplay.

5.4 _includes/scripts.js (explicit initialization fallback)
const carouselEl = document.querySelector('.carousel');
if (carouselEl && typeof Flickity !== 'undefined') {
    const existing = Flickity.data(carouselEl);
    const flkty = existing || new Flickity(carouselEl, {
        setGallerySize: true,
        imagesLoaded: true,
        cellAlign: 'left',
        contain: true,
        autoPlay: 4500,
        wrapAround: true,
        pauseAutoPlayOnHover: false
    });
    flkty.playPlayer();
}
