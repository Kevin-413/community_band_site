Image notes

Use this folder: src/_images
Do not edit _site/_images

File format guide
- Photos: use JPG/JPEG
- Logos with transparency: use PNG
- Graphics/illustrations with transparency: use PNG
- Avoid very large PNG photos (they load slowly)

For slideshow images, keep these names:
band1.jpg
band2.jpeg
band3.png
band4.jpg
band5.jpeg

Best size for slideshow: 1600x900
Best target file size for slideshow: under 500 KB each

General size targets
- Hero/banner photos: 1600px wide max
- Content photos: 1200px wide max
- Small logos/icons: 220px to 500px wide as needed

If you use a new filename, update _data/config.json
